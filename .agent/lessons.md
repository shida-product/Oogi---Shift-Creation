# 教訓メモ

## 2026-04-17: スコアリングのTier固定は穴埋めメンバーを殺す
- **問題**: 目標ありスタッフのベーススコアを固定値(20000)にすると、穴埋めメンバー(10000)に永遠に出番が回らない
- **対策**: 達成率に応じてベーススコアを20000→10000へグラデーション降下させる
- **公式**: `base = 20000 - (completionRate × 10000)`

## 2026-04-17: 他店舗応援の優先度が高すぎると連鎖的に破綻する
- **問題**: 「自店舗半日」より「他店舗フルタイム応援」を先に探すと、応援に出たメンバーの本来の店舗が空く
- **例**: 木庭AM可+諫早フル可 → 諫早を恵比寿に引っ張る → 渋谷が空く → 木庭がAM渋谷に回される
- **対策**: 「他店舗から引っ張る」前に「自店舗の半日」を先に試す（ホーム優先原則）

## 2026-04-17: replace_file_contentの精度
- **問題**: TargetContentが長すぎると、微妙な空白・改行の不一致でマッチしないことがある
- **対策**: 編集前に必ず `view_file` で最新の行番号を確認し、正確なTargetContentを指定する
